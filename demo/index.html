SMILE HACKATHON

Новий формат змагань для ІТ-спеціалістів в Луцьку. Дебютний захід такого формату не матиме конкретної теми. Усім учасникам пропонується представити будь-яку свою ідею чи рішення без обмежень у виборі технологій.

Розклад

  • 09:30 Реєстрація
  • 10:30 Відкриття заходу
  • 11:00 Обговорення ідей та старт роботи
  • 13:00 Обід!
  • 14:00 Робота над розробкою проектів
  • 18:00 Презентації проектів
  • 19:00 Визначення переможців
  • 19:30 Нагородження переможців
  • 20:00 AfterParty!

Для кого ця подія?

У тебе є ідея чи проект, а для втілення потрібно трохи більше натхнення і однодумців

Тобі сюди!

Твій проект може бути побудований на будь-яких платформах та фреймворках – це може бути мобільний додаток, програма, скрипт, веб-додаток, нейронна мережа, тощо.

Умови участі

  • обов’язкова попередня реєстрація
  • наявність персональної техніки для роботи (ноутбук, телефон, тощо)
  • гарний настрій

Що ви отримаєте?

  • можливість реалізувати свої власні ідеї та отримати підтримку серед однодумців
  • комфортне та сучасне місце для роботи
  • можливість самостійно розпоряджатися відведеним часом
  • смачний обід, та безлімітний доступ до чаю та кави
  • Вечірка для святкування успішної реалізації ідеї та старту чогось нового
